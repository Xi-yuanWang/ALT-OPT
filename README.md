# Reproducing Baselines
To reproduce ALTOPT in the original paper.
```
mkdir baseline
chmod +777 runall.sh
./runall.sh
```

To reproduce ALTOPT with K=3,
```
mkdir baselinek3
chmod +777 runall_K=3.sh
./runall_K=3.sh
```
